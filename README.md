## Описание программы

*"Quote Generator - графическое приложение на Python (Tkinter) для генерации случайных цитат.
Включает встроенную коллекцию мотивационных и жизненных цитат от русских классиков, Харуки Мураками и других известных людей.
Позволяет добавлять свои цитаты, просматривать историю генераций и фильтровать её по автору и теме.
Все данные сохраняются в JSON‑файлах, поэтому ничего не теряется при закрытии."*

---

## Требования для запуска

- Python 3.7 или новее
- Дополнительные библиотеки не требуются (tkinter, json, random, os уже есть в Python)

---

## Как запустить

```bash
git clone https://github.com/dekzvtmr/Random_Quote_Generator_GUI.git
cd Random_Quote_Generator_GUI
python quote_generator.py
```

### Как пользоваться
#### Как пользоваться (пошагово)
1. После запуска открывается окно с тремя полями: Текст, Автор, Тема, и кнопками «Добавить цитату» и «Сгенерировать случайную цитату».
2. Добавление своей цитаты:
3. Заполни все три поля.
4. Нажми кнопку «Добавить цитату».
5. Если какое-то поле осталось пустым, появится предупреждение - заполни его и повтори.
6. Генерация случайной цитаты:
7. Нажми кнопку «Сгенерировать случайную цитату».
8. Программа выберет случайную цитату из общей коллекции (встроенные + твои).
9. Новая запись сразу появится в списке истории в нижней части окна.
10. Просмотр истории:
11. Все сгенерированные цитаты отображаются в белом поле.
12. Каждая строка выглядит так: «Текст цитаты» - Автор [тема].
13. Фильтрация истории:
14. Под кнопкой генерации есть два поля: Автор и Тема.
15. Введи в одно из этих полей (или в оба) имя или тему.
16. Нажми «Фильтровать» - в списке останутся только подходящие записи.
17. Можно заполнить только одно поле, фильтр сработает по нему.
18. Сброс фильтров:
    1. Нажми кнопку «Сбросить», чтобы очистить поля фильтра и снова увидеть всю историю.
19. Завершение работы:
    1. Просто закрой окно - все данные уже сохранены в файлах quotes.json и history.json.
    2. При следующем запуске история и коллекция будут на месте.
